# 🔌 Phân Hệ Năng Lực Lõi (Plugins Directory)

> **Khu vực Đóng quân:** ecosystem/plugins/
> Lực lượng Tier-2 Sandbox (Mô-đun Khối Nặng) của OmniClaw.

---

### ⚙️ airllm
> The repository does not contain any readable standard files and might be empty or uses an obscure structure, making it difficult to determine its purpose.

### ⚙️ byterover
> ByteRover Memory Provider is a persistent memory solution using the `brv` CLI for hierarchical knowledge management and retrieval. It integrates with Hermes for configuration and cloud sync options.

### ⚙️ claude_plugin
> Claude Octopus is a plugin that integrates multiple AI providers to provide diverse perspectives on coding and research tasks. It ensures consensus before finalizing outputs through a 75% agreement gate.

### ⚙️ hindsight
> Hindsight Memory Provider is a tool for managing long-term memory with features like knowledge graph and entity resolution. It supports both cloud and local modes of operation.

### ⚙️ holographic
> This repository provides a local SQLite fact store with advanced features such as FTS5 search, trust scoring, entity resolution, and HRR-based compositional retrieval for managing and querying facts in an AI system.

### ⚙️ honcho
> This repository provides an AI-native user modeling solution using the Honcho Memory Provider. It includes tools for semantic search, dialectic Q&A, and persistent conclusions, and can be set up either in a cloud or local environment.

### ⚙️ mem0
> This repository provides server-side LLM fact extraction with semantic search and reranking capabilities using the Mem0 API. It includes setup instructions for integration into an existing Hermes configuration.

### ⚙️ notebooklm_mcp
> Tier-2 Heavy Plugin Module

### ⚙️ omnipets
> This directory contains the master registry, stat generator, and ASCII frame data for all Virtual Companions in the OmniClaw operating system.

### ⚙️ openviking
> This repository provides an OpenViking Memory Provider for the Hermes application, enabling semantic search and knowledge management with a filesystem-like hierarchy.

### ⚙️ retaindb
> This repository provides a cloud memory API with hybrid search capabilities and support for seven different memory types, requiring a RetainDB account and configuration through environment variables or command-line tools.

### ⚙️ supermemory
> The Supermemory Memory Provider is a plugin for Hermes that integrates semantic long-term memory capabilities, including profile recall and explicit memory management tools. It enhances session context with relevant memories and allows for storing, searching, and forgetting memories.
