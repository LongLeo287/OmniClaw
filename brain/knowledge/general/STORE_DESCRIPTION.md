# Chrome Web Store Description

## Short Description (132 characters max)
AI assistant with context menu. Right-click to access AI prompts and chat with OpenAI on any webpage.

## Full Description

**Assistant context** brings AI assistance directly to your browser with seamless context menu integration.

### Key Features

✨ **Right-Click Context Menu**
- Access AI prompts instantly from any webpage
- Customize your own prompts for quick access
- Works on all websites

💬 **Floating Chat Interface**
- Draggable chat popup that works on any webpage
- Continue conversations with full context
- Clean, minimal design

⚙️ **Easy Configuration**
- Set up your OpenAI API key in seconds
- Choose from multiple GPT models (GPT-4, GPT-3.5, GPT-4o, etc.)
- Customize prompts to match your workflow

🔒 **Privacy First**
- All data stored locally in your browser
- API key never leaves your device
- Direct communication with OpenAI API only

### How It Works

1. **Configure**: Click the extension icon to set your OpenAI API key and model preferences
2. **Customize**: Add your own prompts (one per line) for quick access
3. **Use**: Right-click anywhere on a webpage → Select "Assistant context" → Choose a prompt
4. **Chat**: The floating chat interface appears, ready for conversation

### Use Cases

- Quick translations and explanations
- Content summarization
- Code assistance
- Writing help
- Research and Q&A
- And much more!

### Requirements

- Valid OpenAI API key (get one at https://platform.openai.com/api-keys)
- Chrome browser with Manifest V3 support

### Privacy

Your API key and all data are stored locally in your browser. We never collect, store, or transmit your data to our servers. All API calls go directly from your browser to OpenAI.
