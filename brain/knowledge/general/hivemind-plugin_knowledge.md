# HiveMind Context Governance

> **The operating system for AI coding sessions.**

## 🇻🇳 Vietnam Market Priority Release v2.8

HiveMind is an [OpenCode](https://opencode.ai) plugin that helps AI agents not lose context, not forget architectural decisions, and not lose Status: during long sessions. Focus of v2.8: clear onboarding, tight governance, and practical deployment for the Vietnamese team first.

### 10 impressive demo scenarios for launch
1. `SaaS 0→1 for non-coders`: Q&A menu + auto-lane to produce deployable PRDs.
2. `Rescuing chaotic enterprise team prompts`: requirement breakdown, ambiguity map, risk register.
3. `War-room production incident`: force agent to follow evidence checklist before concluding on a fix.
4. `TDD autopilot`: agent automatically transitions from `spec -> build -> validate` with test gates.
5. `MCP-first research sprint`: coordinate Context7/DeepWiki/Tavily/Exa/Repomix and assign confidence scores.
6. `Brownfield modernization`: scan old codebase, set up refactor workflow by lane and checkpoint.
7. `Cross-domain planning`: a single framework for dev + marketing + finance + office-ops.
8. `Subagent swarm governance`: delegate tasks in parallel while retaining trace, export, and historical review.
9. `Bilingual coaching mode`: EN/VI output with the same structure, supporting multi-role team onboarding.
10. `No-command recovery`: user speaks naturally, system automatically realigns to appropriate command and requests permission for next step.
