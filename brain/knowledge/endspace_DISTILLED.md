# Endspace Theme For NotionNext

[![Awesome Arknights Endfield](https://github.com/palmcivet/awesome-arknights-endfield/blob/main/assets/badge.svg?raw=true)](https://github.com/palmcivet/awesome-arknights-endfield)

> 受《明日方舟：终末地》视觉美学启发的 NotionNext 主题。

**Endspace** 是一款风格强烈的 NotionNext 主题，致力于为您的个人博客赋予未来主义与工业科幻的独特气质。其 UI 设计深度致敬了[《明日方舟：终末地》官方网站](https://endfield.hypergryph.com/)的视觉风格。

## ✨ 特性

- **工业科幻设计**：采用严谨的布局、等宽字体及科技感装饰元素。
- **沉浸式加载体验**：包含系统初始化步骤与扫描动效的自定义加载序列。
- **集成媒体播放**：内置音乐播放器组件。
- **响应式布局**：针对桌面端与移动端阅读体验进行了深度优化。
- **丰富的配置项**：提供广泛的选项以自定义颜色、动画及显示元素。

## � 安装与使用

本主题源代码位于 `theme` 分支。

1.  **下载主题**：请下载本仓库 `theme` 分支的所有文件。
2.  **安装**：将下载的文件放置于 NotionNext 项目的 `themes` 文件夹下，并重命名为 `endspace`（即路径应为 `themes/endspace`）。
3.  **安装依赖**：Endspace 主题依赖 `@tabler/icons-react` 和 `remixicon-react` 图标库。请确保在您的 NotionNext 项目根目录的 `package.json` 中包含以下依赖：
   ```json
   "@tabler/icons-react": "^3.36.1",
   "remixicon-react": "^1.0.0"
   ```
   然后运行 `npm install` 或 `yarn install` 来安装依赖。
4.  **启用**：修改 NotionNext 的环境变量或配置文件，将 `THEME` 设置为 `endspace` 即可生效。

# Endspace Theme For NotionNext

[![Awesome Arknights Endfield](https://github.com/palmcivet/awesome-arknights-endfield/blob/main/assets/badge.svg?raw=true)](https://github.com/palmcivet/awesome-arknights-endfield)

> A NotionNext theme inspired by the visual aesthetics of *Arknights: Endfield*.

**Endspace** is a highly stylized NotionNext theme designed to bring a unique futuristic and industrial sci-fi atmosphere to your personal blog. Its UI design pays deep homage to the visual style of the [official *Arknights: Endfield* website](https://endfield.hypergryph.com/).

## ✨ Features

- **Industrial Sci-Fi Design**: Rigorous layout, mono-spaced typography, and tech-inspired decorative elements.
- **Immersive Loading Experience**: Custom loading sequence including system initialization steps and scanning effects.
- **Integrated Media Playback**: Built-in music player component.
- **Responsive Layout**: Deeply optimized for both desktop and mobile reading experiences.
- **Rich Configuration Options**: Extensive settings to customize colors, animations, and display elements.

## 🚀 Installation & Usage

The source code for this theme is located in the `theme` branch.

1.  **Download Theme**: Please download all files from the `theme` branch of this repository.
2.  **Install**: Place the downloaded files into the `themes` folder of your NotionNext project and rename the folder to `endspace` (so the path should be `themes/endspace`).
3.  **Install Dependencies**: The Endspace theme depends on the `@tabler/icons-react` and `remixicon-react` icon libraries. Make sure to include the following dependencies in your `package.json` at the root of your NotionNext project:
   ```json
   "@tabler/icons-react": "^3.36.1",
   "remixicon-react": "^1.0.0"
   ```
   Then run `npm install` or `yarn install` to install the dependencies.
4.  **Enable**: Modify the NotionNext environment variables or configuration file and set `THEME` to `endspace`.

# Endspace Theme For NotionNext

[![Awesome Arknights Endfield](https://github.com/palmcivet/awesome-arknights-endfield/blob/main/assets/badge.svg?raw=true)](https://github.com/palmcivet/awesome-arknights-endfield)

> 『アークナイツ：エンドフィールド』の視覚的な美学にインスパイアされたNotionNextテーマ。

**Endspace**は、個人ブログに未来主義とインダストリアルSFの独特な雰囲気をもたらすことを目指した、非常にスタイリッシュなNotionNextテーマです。そのUIデザインは、[『アークナイツ：エンドフィールド』公式サイト](https://endfield.hypergryph.com/)のビジュアルスタイルに深く敬意を表しており。

## ✨ 特徴

- **インダストリアルSFデザイン**：厳格なレイアウト、等幅フォント、およびテクノロジー感のある装飾要素を採用。
- **没入型ローディング体験**：システム初期化ステップとスキャンアニメーションを含むカスタムローディングシーケンス。
- **統合メディア再生**：音楽プレーヤーコンポーネントを内蔵。
- **レスポンシブレイアウト**：デスクトップおよびモバイルでの閲覧体験向けに深く最適化されています。
- **豊富な設定項目**：色、アニメーション、表示要素をカスタマイズするための広範なオプションを提供。

## 🚀 インストールと使用法

本テーマのソースコードは `theme` ブランチにあります。

1.  **テーマのダウンロード**：本リポジトリの `theme` ブランチからすべてのファイルをダウンロードしてください。
2.  **インストール**：ダウンロードしたファイルをNotionNextプロジェクトの `themes` フォルダに配置し、フォルダ名を `endspace` に変更してください（パスは `themes/endspace` となります）。
3.  **依存関係のインストール**：Endspaceテーマは `@tabler/icons-react` と `remixicon-react` アイコンライブラリに依存しています。NotionNextプロジェクトのルートの `package.json` に以下の依存関係が含まれていることを確認してください：
   ```json
   "@tabler/icons-react": "^3.36.1",
   "remixicon-react": "^1.0.0"
   ```
   その後、`npm install` または `yarn install` を実行して依存関係をインストールしてください。
4.  **有効化**：NotionNextの環境変数または設定ファイルを変更し、`THEME` を `endspace` に設定することで有効になります。
