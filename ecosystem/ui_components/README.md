# 🎨 OmniClaw Frontend Components (UI Factory)

> **The Blank Canvas for AI-Generated React/Tailwind Assets.**
> System Integrity Level: V5.0 Authenticated

Welcome to the **UI Components** domain. Unlike the OS Core or the Native Tool registries, this directory is specifically designed as a **Passive Outputs Repository**.

Whenever OmniClaw Agents (specifically Frontend Web Developer Sub-agents) are executed to build visual interfaces (like a local Dashboard or a web app for the User), all their `.tsx`, `.jsx`, and `.css` logic must be dumped here.

---

## 🏛️ Operating Protocol

1. **No Backend Code Allowed**: If you are writing Python integration scripts, MCP configurations, or OS-level heuristic crawlers, **HALT IMMEDATELY**. Move that code to `ecosystem/tools/`.
2. **No Abstract Prompts Allowed**: If you are writing a standard Markdown guideline (e.g., "How to use Shadcn"), route it to `ecosystem/skills/`.
3. **Atomic Component Storage**: When downloading Shadcn UI, magic-ui, or aceternity components, organize them into standard `components/ui` subdirectories.
