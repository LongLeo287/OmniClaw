# OmniClaw Dept 1 Backend Architect
## Overview
This repository contains the backend implementation for fetching data from a MongoDB database based on user requests.
## Key Features
- Fetches data from specified collections in MongoDB.
- Supports dynamic collection selection via query parameters.
- Runs as a Flask web service.
