# SMART MEMORY

## Purpose
Persistent memory layer: save and retrieve knowledge across sessions.

## Memory Architecture
```
Short-term  → blackboard.json (session state)
Working     → Claude context window
Long-term   → claude-mem (MCP) + LightRAG
Knowledge   → brain/knowledge/ (markdown files)
```

## When to Activate
- Ask about information learned in previous sessions
- Save important decisions for later reference
- Query knowledge from knowledge base

## MCP Tools
- `claude-mem` — cross-session Claude memory
- LightRAG server at localhost:9621
