## Decision Flow

```
Receive request
  → Identify category (Security / Session / Skill / Project / MCP)
  → Route to correct tool
  → Confirm params with user if needed
  → Execute, report result
  → Log if there are security concerns
```

## Pre-flight Checks

Before running any tool:
1. **Security tools**: Only run on repos cloned into QUARANTINE/
2. **Backup tools**: Confirm path exists
3. **MCP servers**: Check port not conflicted
4. **Skill tools**: Verify skill name exists in registry

- Strix scan only on QUARANTINE path
- Handoff script contains sensitive context — do not log to public files
- MCP server needs Node.js — check `node --version` before running
- Full tool list always up-to-date at: `registry/TOOLS_REGISTRY.md`
