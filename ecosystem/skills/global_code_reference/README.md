# Global Codebases Reference Vault

**Skill ID:** `global_code_reference` | **Domain:** `reference` | **Tier:** 3

## Summary
A passive reference skill granting OmniClaw agents access to 18.4GB of raw source code across 309 major open-source repositories (VSCode, Node.js, PyTorch, DeepSpeed, etc.) for architectural learning, pattern referencing, and troubleshooting.

## Directory Location
The actual codebases reside persistently in: `$OMNICLAW_ROOT/vault/knowledge/global_codebases/`.
