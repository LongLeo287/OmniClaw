# 🧠 OmniClaw Skills Ecosystem

> **Passive Cold Storage & Competency Warehouse**  
> Total Authorized Skills: 1,932+

Welcome to the central repository for OmniClaw's semantic capabilities and subagent personas.

This directory represents the **Omnibus Assimilated Brain**. It houses over 1,900 discrete "Skills" (ranging from technical stacks like React and Docker, to security protocols like AWS Pentesting, to contextual Personas).

## 🛑 Zero-Trust "Cold Storage" Architecture
Just like the `ecosystem/plugins` directory, the `ecosystem/skills` zone is **exclusively Passive**.
These skills do not execute automatically. They are raw text documents and machine-readable `schema.json` files. 

**Execution Policy:**
1. Core Daemons (like the Orchestrator, oiw_intake) dynamically "read" a skill's directory when a task demands it.
2. The skill acts as an injection payload, giving the Subagent immediate subject-matter expertise.
3. No rogue background processes are spawned.

## 📋 Standard Validation (`SKILL_SPEC.md`)
Every single folder in this directory has undergone the **V5.0 Assimilation Pipeline**.
This means every skill possesses:
1. `SKILL.md` - The actual Prompt/Payload logic.
2. `schema.json` - The machine-readable identity payload making it queryable by the system.
3. `README.md` - The human-readable competency profile bearing the official OmniClaw Watermark.

All integrated operations are statically listed in the `SKILL_REGISTRY.json` file. Do not manually edit this JSON file; it is compiled via the Assimilation Pipeline.
