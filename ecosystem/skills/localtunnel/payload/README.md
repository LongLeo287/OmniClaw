# OmniClaw Backend
This project serves as the backend component of the OmniClaw system, integrating with localtunnel on port 114458.
## Requirements
- Flask
- Requests
- Django (for dependency management)
## Running the Application
To run this application, ensure you have Python installed. Then follow these steps:
1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `python main.py`
3. Access it via localtunnel or a similar service.
