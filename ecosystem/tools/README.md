# 🔧 OmniClaw Tools Ecosystem

> **Active Registry for Physical Scripts & APIs**

This directory is the definitive registry for actual, executable utility scripts, MCP (Model Context Protocol) modules, APIs, and binary wrappers used across the OmniClaw infrastructure.

Unlike `plugins` or `skills` (which are strictly passive "Cold Storage"), the items in `tools` are physical scripts designed to be executed by the system or subagents (such as basic heuristic survival modules, pingers, or directory scanners).

## Navigation
- Check `_REGIONAL_MAP.md` for a complete index of all registered toolsets.
- Check `tool_template.md` for the standard template required when submitting new scripts to this directory.
